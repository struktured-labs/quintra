# Quintra v0.18.65 — Unfolding Compass

SELECT is now an exploration-first tile map. It begins with the current room
and one hollow reachable frontier, then fills the 6×5 dungeon graph with the
route actually walked. Deeper rooms and links stay blank; earned Sigil, trial,
boss, and Rift Well clues retain their distinct colors and symbols.

Native mGBA training checkpoints can now be captured from one continuous
controller run every five minutes. The six default Picsean Easy `.ss0` files
are ROM/controller/generator hash-bound and independently cold-load verified.

Easy mode's Void Lord World Collapse now deals one damage, matching Easy's
inspection contract. Normal retains the full room-wide positional blast.

The stage-scale audit also records a remaining design gap: 20–30 one-screen
rooms can still feel compact. Continuous outdoor spaces, stronger geographic
branch identity, and genuinely scrolling Penta-style boss arenas remain
future engine and content work.
