# Quintra v0.18.72 — The Sigil Key

Quintra now explains its navigation controls before a run begins. Champion
selection plainly says that Select opens the map and Start opens the Pack,
without adding another modal or delaying repeat runs.

The Pack now answers “what do I do next?” from the same persisted progression
state that gates the dungeon. Its live quest line identifies the Rift Sigil as
the stage key, then advances through the Warden, Waystone, Deep Ward, and
marked skull gate. Riftwild, villages, and Colossus arenas receive their own
direct guidance.

The guidance survives backtracking, nonlinear Rift Wells, suspend/resume, and
deep emulator checkpoints because it does not infer progress from the current
room. A linked-ROM regression verifies the two control prompts and all eight
quest states directly from the native Game Boy tilemap.

This release retains v0.18.71's paired 224×200 dungeon wings, 20–30-room
procedural stages, 460 PyBoy and native mGBA deep checkpoints, six five-minute
training states, nine screen-scale Colossi, and identical Normal/Easy content.
