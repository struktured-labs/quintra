# Quintra v0.18.71 — Dungeon Horizons

Dungeon scale now changes in play, not only on the Compass. Every complete
snake row ends in a generated two-room wing: dense approach expanses
4/10/16/22 lead into lighter turn courts 5/11/17/23. Both are 224×200 halls
with a real two-axis camera. Stage one has six scrolling dungeon fields and
late stages have eight. The old 160×136 viewport edges are interior terrain;
reciprocal doors live on the far 28×25 perimeter.

Each court contains seed-shifted ruin halls, pillar clusters, stage-colored
accents, clear cardinal trails, and encounters in the added eastern and
southern field. Rendering, collision, projectiles, enemy AI, controller
routing, camera state, and doors all share the same coordinates. Important
fixtures retain their authored roles, so the dungeon alternates compact
puzzle/combat chambers with three or four sustained two-room expedition wings.

The new `court` deep-test family covers all nine stages, five champions, and
both difficulties. PyBoy and native mGBA curricula now contain 460 ROM-bound
states; all 230 Normal/Easy pairs retain identical geometry and encounters.
The native smoke set independently cold-loads all six checkpoint families.

Live-ROM coverage crosses both former room edges, reaches the southeast camera
bound, proves every real graph door connected, verifies far-field enemies and
architecture, crosses one wide-to-wide threshold, exits into an ordinary
160×136 room without leaked camera state, and re-enters from the south at the
correct far edge. The release reel and dungeon still now show the live
scrolling court.
